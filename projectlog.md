# 开发日志

## 2026-04-18 项目初始化

### 需求
为大学生求职季设计一个看板式申请管理工具，帮助用户清晰掌控多个公司的申请进度、截止日期、材料提交状态。

### 技术选型
- React 18 + TypeScript + Vite
- @dnd-kit 实现拖拽
- CSS Modules 样式隔离
- LocalStorage 数据持久化

### 已完成功能
- 6 列看板布局（待投递、已投递、笔试、面试中、已录用、已拒绝）
- 卡片拖拽移动，跨列自动更新状态
- 申请的新增、编辑、删除（Modal 表单）
- 材料提交清单及进度条
- 截止日期到期提醒（3天内黄色、过期红色）
- 公司/岗位搜索过滤
- 顶部各阶段数量统计
- LocalStorage 自动保存
