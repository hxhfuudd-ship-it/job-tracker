# 求职申请管理看板

大学生求职季申请进度管理工具，Trello 风格看板界面。

## 功能

- 6 阶段看板：待投递 → 已投递 → 笔试 → 面试中 → 已录用 / 已拒绝
- 拖拽卡片在列间移动，自动更新状态
- 新增 / 编辑 / 删除申请（公司、岗位、截止日期、薪资、备注）
- 材料提交清单及进度追踪
- 截止日期提醒：3 天内黄色警告，已过期红色标记
- 按公司名或岗位搜索过滤
- 顶部数据统计栏
- LocalStorage 自动持久化，刷新不丢失

## 技术栈

- React 18 + TypeScript + Vite
- @dnd-kit（拖拽）
- CSS Modules
- LocalStorage

## 启动

```bash
npm install
npm run dev
```
